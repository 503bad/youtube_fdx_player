# Yotube XFD Player
Youtubeにて公開したクロスフェード動画をスマホなどで再生しやすくするためのプレイヤーです。
Vue.js＋Vite環境で動作します。

public/albums.json にトラックの情報を記載するとプレイヤーで再生できるようになります。
デモサイトはコチラ：https://player.date50.net/
